# 🎓 CET-6 Tutor — 六级金牌私教

Hermes Agent skill：英语六级备考助手，支持阅读真题、听力练习、艾宾浩斯记忆背单词。

## 快速安装

```bash
hermes skills tap add https://kkgithub.com/qingzhou-hub/cet6-tutor
hermes skills install cet6-tutor
```

## 功能

- 📖 **阅读真题** — 96篇完整六级阅读（2018-2025年，23套试卷）
- 🎧 **听力真题** — 27套完整听力试题（音频嵌入HTML）
- 📝 **单词背诵** — 艾宾浩斯记忆引擎，7级状态管理
- ✅ **自动批改** — 提交答案即出分

## 数据来源

真题数据来自 AstrBot 插件 [astrbot_plugin_cet6](https://github.com/202704948-design/astrbot_plugin_cet6)。
